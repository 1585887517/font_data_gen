# Text Corpus for Dataset Generation

Place open-source or public-domain text files in this folder to enrich document content.

Accepted file extensions:
- `.txt`

Each non-empty line will be treated as a text sentence or phrase that can be used by the generator.

Examples:
- invoice text
- contract clauses
- form fields
- report headlines
- public notices

Do not place proprietary or copyrighted text unless you have permission to use it.
